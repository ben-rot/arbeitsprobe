# Arbeitsprobe
Kleines angefangenes Projekt


## Tech Stack:
- Go für Backend
- Templ, HTMX & AlpineJS für Frontend
- Tailwind für Styling
- Postgres als Datenbank


## Problem versucht zu lösen:
- Organisation der Clankriegsliga für Spieler & Clanfamilien
- Anmeldung von Spieler-Accounts bei Clanfamilien + Übersicht mit Status
- Zuordnen der angemeldeten Spieler auf Clans der Clanfamilie durch Manager
- Integration mit Discord-Bot zur Automatisierung von Prozessen


## Status des Projektes
- Fertige UI ohne Responsiveness
- SQL Schemas und vereinzelte Queries
- Routes & Handlers für vereinzelte API-Endpoints (überwiegend Page-Routing)
- Grobe Middleware, Authentifizierung per Discord OAuth2
- Grundlegende Funktionalität fehlend


## Transparenz
Projekt wurde mit externer KI (Gemini, ChatGPT free) als Hilfe & Entblocker erarbeitet
